1. การติดตั้งฮาร์ดแวร์ (Hardware Setup)

โปรดติดตั้งตามลำดับเพื่อความเสถียรของสัญญาณที่ยอดเยี่ยมที่สุด

01

Wireless Keyboard

เสียบ USB Receiver เข้ากับ สายพ่วง USB สีน้ำเงิน ก่อนนำไปเสียบที่เครื่อง เพื่อลดการกวนสัญญาณจากความร้อน

02

AV & Display

ต่อสาย HDMI เข้าจอ TV 4K และเสียบ USB Sound Card โดยใช้ช่องสัญญาณ สีเขียว เพื่อส่งเสียงไปยังแอมป์

03

Power System

เชื่อมต่อ Power Adapter และกดปุ่มเปิดเครื่องเพื่อเริ่มขั้นตอน Initial Activation บนหน้าจอ

2. การเปิดใช้งานครั้งแรก (Initial Activation)

ขั้นตอนการลงทะเบียนผ่านระบบ Cloud เพื่อความสนุกที่สมบูรณ์แบบ

🌐

Configure Network

Setting > Network & Internet (แนะนำใช้สาย LAN เพื่อความนิ่งสูงสุด)

📸

Line Activation

สแกน QR Code บนหน้าจอผ่านแอป Line เพื่อยืนยันลิขสิทธิ์และเปิดใช้งานเครื่อง

☁️

Cloud Sync

ปรับสถานะเป็น "On" เพื่อให้เครื่องเริ่มโหลดฐานข้อมูลเพลงล่าสุดมายังตัวเครื่อง

✅ สถานะการทำงานปกติ

ความพร้อมของระบบสังเกตได้ง่ายๆ จาก "เวลา" บนหน้าจอเครื่อง หากเวลาเป็นปัจจุบัน หมายความว่าการเชื่อมต่ออินเทอร์เน็ตสมบูรณ์ 100%

🛡️ Technical Measures

  • Time Sync: หาก Activate ไม่ผ่าน โปรดเช็คการตั้งค่าวันที่และเวลาให้ตรงกับปัจจุบัน
  • Ground Noise: หากเกิดเสียงจี่เมื่อต่อแอมป์ใหญ่ ให้ใช้สายสัญญาณคุณภาพสูง หรือ Ground Loop Isolator
  • LAN Priority: ปัญหาแอปหาเครื่องไม่เจอ 90% เกิดจาก Wi-Fi แนะนำให้ใช้ สาย LAN เป็นหลัก
